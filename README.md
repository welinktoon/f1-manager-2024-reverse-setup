# F1 Manager 2024 Reverse Setup Helper

Мини-калькулятор для настройки болида в F1 Manager 2024.

## Для чего

В игре есть пять итоговых характеристик настройки:

- Избыточная поворачиваемость
- Устойчивость при торможении
- Повороты
- Сцепление с поверхностью
- Прямые участки

Этот инструмент работает от обратного: пользователь двигает пять итоговых меток туда, где хочет видеть их в игре, а калькулятор подбирает значения:

- Угол переднего крыла
- Угол заднего крыла
- Anti-roll / настройки уменьшения
- Развал колес
- Расхождение колес

Модель основана на открытых формулах F1 Manager setup calculator. Из-за игровых округлений и скрытых нюансов точность не гарантирует 100%, но для F1 Manager 2024 обычно помогает попасть в нужные диапазоны примерно на 90% и быстрее довести сетап до топового.

## Как пользоваться

1. В F1 Manager 2024 открой настройки болида.
2. Посмотри, где находятся оптимальные бирюзовые диапазоны.
3. На странице выставь пять итоговых ползунков примерно в центр этих диапазонов.
4. Нажми **Посчитать настройки**.
5. Перенеси лучший вариант в игру.

## Локальный запуск

Открой `index.html` в браузере. Сервер не нужен.
